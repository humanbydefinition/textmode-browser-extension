# Support

This project is maintained on a best-effort basis. Please choose the route that matches what you need.

## Questions

Use GitHub Discussions once they are enabled, or open a question issue if Discussions are unavailable.

Good questions include:

- How to build or load the extension locally.
- Whether a page or media type should be supported.
- How extension behavior relates to textmode.js.
- Whether an idea should become a feature request.

## Bug Reports

Use the bug report issue template.

Helpful reports include:

- Browser and version.
- Operating system.
- Extension build target.
- Steps to reproduce.
- Expected and actual behavior.
- Whether the media is canvas, video, cross-origin, local, streaming, or protected.
- Screenshots, recordings, or console output when useful.

Please remove private page content, credentials, tokens, proprietary media, and personal data before posting.

## Feature Requests

Use the feature request template.

For features that affect permissions, storage, browser support, exports, or the runtime architecture, explain
the user workflow and why the current behavior is not enough.

## Security Reports

Do not report security issues publicly. Follow [SECURITY.md](SECURITY.md).

## textmode.js Help

For textmode.js library usage, start with:

- [code.textmode.art](https://code.textmode.art)
- [editor.textmode.art](https://editor.textmode.art)
- [github.com/humanbydefinition/textmode.js](https://github.com/humanbydefinition/textmode.js)

If a problem only happens through this extension, report it here. If it reproduces in a plain textmode.js
sketch, report it upstream in the library project.
